Data products from the database contents

To facilitate analyses, a variety of data products, typically rectangular, matrix-like, summarised contents of information from the database were constructed. This include summarised timeseries for a subset of metrics (e.g. SSB, TB, R, TC and F) as well as timeseries which fulfill certain criteria (e.g. all SR pairs with years of data). These products were assembled as database views using the Structured Query Language (SQL).
